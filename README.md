# Mood Tracker (Flask + Jinja Website)

A simple Flask web application that allows users to record their current mood and view previously submitted moods with timestamp.  
This assignment demonstrates the use of Flask routing, form handling, and dynamic HTML rendering using Jinja templates.

----

## Instructions to be run

### Step 1: Create Virtual Environment

python3 -m venv venv
source venv/bin/activate     # For macOS/Linux
venv\Scripts\activate      # For Windows

### Step 2: Install Dependencies

pip install -r requirements.txt

### Step 3: Run the Application

python3 app.py

### Step 4: Open in Browser

Go to:  **http://127.0.0.1:5000/**

----

## Requirements

Flask==3.0.3

----

## Features Implemented

- User form to input current mood  
- Flask routes for form submission and display  
- Dynamic mood list rendered with Jinja2  
- CSS styling
